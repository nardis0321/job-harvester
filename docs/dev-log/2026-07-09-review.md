### 추가

#### `src/collectors/base.py`
collector 공통 인터페이스. `collect(query, max_pages) -> List[JobPosting]` 형태를 정의.

#### `src/collectors/saramin.py`
핵심 구현. 사람인 검색 페이지 요청, HTML parsing, 필드 추출, JobPosting 정규화를 담당.

#### `tests/test_collectors.py`
사람인 fixture HTML parser 테스트와 mock source 유지 테스트를 추가.

#### `tests/test_dedupe.py`
`rec_idx`가 다른 URL은 서로 다른 공고로 남는지 테스트를 추가.

### 변경

#### `src/collectors/mock.py`

기존 mock collector를 BaseCollector 인터페이스에 맞게 변경. 기능 자체는 거의 그대로.

#### `src/main.py`
CLI 인자를 추가.`--source`, `--query`, `--max-pages`를 받아 collector를 선택.

#### `src/dedupe.py`
사람인 URL의 `rec_idx`를 dedupe key에 남기도록 수정. 기존처럼 query를 전부 제거하면 사람인 공고 URL들이 모두 같은 URL처럼 취급되는 문제가 있었음. 쿼리 처리 로직 추가.

#### `README.md`
새 실행 방법, collector 목록, 사람인 collector 제한사항을 문서화.
